# Profile 术语表

## 词条

`profile`
: 关于用户、学习目标和协作偏好的当前可信上下文，且这些内容跨目录仍然有效。在 canon 中由 `wiki/profile/` owning。

`current profile`
: 当前有效、跨目录仍然成立的用户画像和偏好集合。

`communication preference`
: 和用户交流时跨目录有效的沟通规范，例如语言、语气、术语处理、回答定制化。

`environment preference`
: 用户跨目录有效的工具环境偏好，例如 terminal、editor、knowledge management。

`learning profile`
: 用户当前身份、技术栈、学习方向、研究方向和学习目标的摘要。

`writing preference`
: 用户对文档写作方式的跨目录偏好，例如文档单一职责、metadata、结构化正文、语言规则。
