# Normative Keyword

## 理论使用

`normative keyword` 的词义边界由 [Agent 术语表](../../vocabulary/agent/index.md) owning。本页只说明 normative keyword 在 agent-facing 文档理论中的作用。

它让 agent 区分不可违反的约束、默认策略和开放空间，避免所有要求都被同一种语气抹平。

## 关系

[Normative Keyword Ladder](../patterns/normative-keyword-ladder.md) 是使用 normative keyword 的模式。

[Strong constraints should be scarce](../claims/strong-constraints-should-be-scarce.md) 限制强约束的使用范围。
