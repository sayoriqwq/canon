# Concepts 概念

Concept 页面定义并连接重要理论对象。

## 概念

- [Document Module](document-module.md)：把 Markdown 文档视为可索引、可链接、可组合、可复用的模块单元。
- [Document Metadata](document-metadata.md)：文档级别的持久状态信息，通常放在 frontmatter 中。
- [Effectful Function Signature](effectful-function-signature.md)：通过函数签名暴露成功值、失败通道和依赖能力的工程边界。
- [断言 Assertion](assertion.md)：说明 assertion 在文档维护理论中的作用。
- [Agent-facing Pattern](agent-facing-pattern.md)：面向 agent 的稳定输入或输出结构。
- [Normative Keyword](normative-keyword.md)：说明 normative keyword 在 agent-facing 文档理论中的作用。
