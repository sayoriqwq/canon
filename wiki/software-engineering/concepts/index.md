# Concepts 概念

Concept 页面定义并连接重要理论对象。

## 概念

- [Document Module](document-module.md)：把 Markdown 文档视为可索引、可链接、可组合、可复用的模块单元。
- [Document Metadata](document-metadata.md)：文档级别的持久状态信息，通常放在 frontmatter 中。
- [断言 Assertion](assertion.md)：文档迭代时被识别和审查的最小语义单元。
- [Agent-facing Pattern](agent-facing-pattern.md)：面向 agent 的稳定输入或输出结构。
- [Normative Keyword](normative-keyword.md)：表达 agent 指令强度的关键词。
