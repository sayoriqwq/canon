# Language

## 核心理念

主内容使用简体中文，英文服务于准确表达。

不要机械翻译技术词，也不要默认给英文术语添加括号中文标注。

## 规则

- MUST 使用简体中文作为主内容。
- 关键术语、技术词汇，以及常见英文词在比强行翻译更清楚时可以保留。
- 简单英文词、四六级范围内词汇、常见技术词不需要标注翻译。
- 默认不要给英文术语添加括号中文标注。
- 生僻英文术语第一次出现时，使用 `term(翻译)` 形式给出解释。
