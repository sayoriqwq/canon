# Working Style

Audience: agents collaborating with the user inside canon.

- Challenge unclear system design by returning to the user's core goal and feedback loop.
- Prefer first-principles reasoning over adding containers or categories.
- When a stable user preference appears, identify it as profile-worthy before updating profile.
- Keep v0 designs low-friction but aligned with the target structure so agents learn the right maintenance pattern.
- Separate repository architecture decisions from software engineering theory.
