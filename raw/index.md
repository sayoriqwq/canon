# Raw 原始材料

`raw/` 按材料形态保存原始材料。它是 source layer，不是 wiki。

Raw 服务来源保真，不服务解释。Agent MUST NOT 改写、翻译、摘要化或重排 raw 正文。

Raw 可以被任意 sub-wiki 的 source note 引用。解释、综合、冲突处理和理论化写入 `wiki/` 下的 owning sub-wiki。
