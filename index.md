# Canon

Canon 是个人软件工程理论系统。先从这里进入，再根据任务进入对应子系统。

## 系统

- [Profile](profile/index.md)：当前用户画像和协作模型。
- [Vocabulary](vocabulary/index.md)：canon 共享的分层术语。
